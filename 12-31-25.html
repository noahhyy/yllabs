New Commands Added

December 31, 2025

Posted by: noah

We’ve added new moderation and utility features to improve overall functionality and control.

/say Command

The /say command has been added, allowing administrators to make the bot send messages directly in channels. This helps with announcements, formatting, and clean automated messaging.

/infract Command

The /infract command is now available for issuing infractions. It supports punishment type, reason, appealable status, and staff notes, making moderation actions clear and consistent.

What’s Next

These additions are just the start. More features, improvements, and refinements are already in development and will be released soon.
